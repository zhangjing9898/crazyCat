# 基于egret的游戏 —— 围住神经猫

小白一个，所以先去找了些资源，跟着练练手。

### 启动项目 

```cli
git clone xxx
egret build xxx(project name)
egret startserver xxx(project name) -a
// 加-a参数是实现自动部署
```
